CSS Essentials Reference Guide

A comprehensive guide to CSS concepts you must know to be confident in CSS development.

1. CSS Selectors

Understanding how to target elements is fundamental:

/* Element */ h1 { color: blue; } /* Class */ .highlight { background: yellow; } /* ID */ #header { font-size: 24px; } /* Descendant */ .nav a { text-decoration: none; } /* Child */ .nav > li { display: inline; } /* Pseudo-classes */ a:hover { color: red; } /* Attribute */ input[type="text"] { border: 1px solid #ccc; }

This text is highlighted using a class selector.

5. Responsive Design

Making websites work on all devices:

/* Mobile first approach */ .container { width: 100%; } @media (min-width: 768px) { .container { width: 750px; } } @media (min-width: 1024px) { .container { width: 1000px; } }

6. CSS Variables

Reusable values throughout your stylesheet:

This uses CSS custom properties!
:root { --primary-color: #007bff; --spacing: 1rem; } .element { color: var(--primary-color); padding: var(--spacing); }

7. Positioning

Control where elements appear:

  • Static: Normal document flow (default)
  • Relative: Positioned relative to normal position
  • Absolute: Positioned relative to nearest positioned parent
  • Fixed: Positioned relative to viewport
  • Sticky: Switches between relative and fixed

8. Animations & Transitions

.button { transition: all 0.3s ease; } .button:hover { transform: translateY(-2px); box-shadow: 0 4px 8px rgba(0,0,0,0.2); }

9. Common Layout Patterns

Centered content using Flexbox
/* Center anything with flexbox */ .center { display: flex; justify-content: center; align-items: center; height: 100vh; } /* Or with grid */ .center-grid { display: grid; place-items: center; }

Key CSS Concepts to Master

  • Selectors: Element, class, ID, pseudo-classes, attribute selectors
  • Box Model: Content, padding, border, margin, box-sizing
  • Display: block, inline, inline-block, flex, grid, none
  • Positioning: static, relative, absolute, fixed, sticky
  • Flexbox: justify-content, align-items, flex-direction, gap
  • Grid: grid-template-columns/rows, grid-area, gap
  • Responsive: Media queries, mobile-first approach
  • Typography: font-family, font-size, line-height, text-align
  • Colors: hex, rgb, rgba, hsl, hsla values
  • Animations: transitions, keyframes, transform
  • Variables: CSS custom properties for maintainable code
  • Accessibility: Focus styles, screen reader support
